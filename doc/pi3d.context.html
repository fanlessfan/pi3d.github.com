Light Module¶

class pi3d.context.Light.Light(lightpos=(10, -10, 20), lightcol=(1.0, 1.0, 1.0), lightamb=(0.10000000000000001, 0.10000000000000001, 0.20000000000000001))[source]¶

Bases: echomesh.util.DefaultInstance.DefaultInstance

Holds information about lighting to be used in shaders

set light values. These are set in Shape.unif as part of the Shape constructor. They can be changed using Shape.set_light() The pixel shade is calculated by componet multiplying lightcol by the texture by the dot product of direction and -normal + component multiplying lightamb by the texture

Arguments: lightpos – tuple (x,y,z) vector direction from the light lightcol – tuple (r,g,b) defines shade and brightness lightamb – tuple (r,g,b) ambient lighting multiplier
